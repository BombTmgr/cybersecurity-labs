# Python Network Security Toolkit

## Overview

This project is a Python-based network security toolkit developed as part of my Master's coursework. It provides a menu-driven interface for performing basic penetration testing tasks.

## Features

* Ping Test
* Port Scanning
* Route Tracing
* User-friendly menu interface
* Error handling for invalid inputs

## Tools & Technologies

* Python
* Networking concepts (TCP/IP)

## How It Works

The program provides a menu where users can select different network tools. Each function is implemented using modular programming for better structure and readability.

## Learning Outcomes

* Developed understanding of network protocols
* Gained hands-on experience in penetration testing techniques
* Improved Python programming skills
* Implemented modular and user-friendly design

## How to Run

1. Clone the repository
2. Run the script:

   ```bash
   python Assessment.py
   ```
